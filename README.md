# 😷 AI Face Mask Detection

A real-time Face Mask Detection System built using **TensorFlow**, **MobileNetV2**, **MediaPipe**, and **OpenCV**. The system detects faces from a webcam and classifies whether a person is wearing a face mask with high accuracy.

---

## 🚀 Features

- 🎥 Real-time webcam detection
- 😷 Face mask / No mask classification
- 📊 Confidence percentage
- 👤 Multiple face detection
- ⚡ FPS counter
- 🎨 Professional UI with bounding boxes
- 🧠 MobileNetV2 Transfer Learning
- 📈 High accuracy model (~98%)

---
